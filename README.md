### React + TailwindCSS + GSAP Animated Cocktail Landing Page

Tipsy Tales is a modern cocktail and bar-style landing page built with **React + TailwindCSS**, featuring smooth **GSAP animations** like scroll-triggered reveals, pinned sections, parallax effects, and timeline-based transitions.

---
## Live Preview
### View the live site here: **[Tipsy Tales](https://tipsytales-kappa.vercel.app/)**

---

## Features

- Modern cocktail/bar UI with clean layout and responsive design  
- GSAP-powered animations:
  - Scroll-triggered transitions (**ScrollTrigger**)
  - Smooth section reveals and timelines
  - Parallax-like motion effects
  - Video/element animations
- Fully responsive (desktop, tablet, and mobile)
- Reusable components structure:
  - `Navbar`, `Hero`, `Cocktails`, `Menu`, `About`, `Art`, `Contact`

---

## Tech Stack

- **React** - JavaScript library for building user interfaces
- **Tailwind CSS** - Utility-first CSS framework
- **GSAP** - Professional-grade animation library
- **Vite** - Next-generation frontend tooling

## Setup + Run + Build

### 1. Clone the repository

```bash
git clone https://github.com/your-username/Tipsy_Tales.git
cd Tipsy_Tales
```

### 2. Install dependencies

```bash
npm install
```

### 3. Run locally (Development)

```bash
npm run dev
```

Open in browser:
```
http://localhost:5173
```

### 4. Build for Production

```bash
npm run build
```

### 5. Preview Production Build

```bash
npm run preview
```

## GSAP Animations

The website features various GSAP animations including:

- **ScrollTrigger**: Elements animate as you scroll through sections
- **Timeline Animations**: Coordinated sequences of animations
- **Parallax Effects**: Elements move at different speeds for depth
- **Pinned Sections**: Sections that stay fixed during scroll
- **Stagger Effects**: Sequential animations across multiple elements
- **Smooth Transitions**: Fluid motion between states
